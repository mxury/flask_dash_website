Predicting crypto prices using a hybdrid stacked model consisting of RNNs and gradient boosted decision trees

Cryptocurrencies have been a new and exciting asset class that has taken the world by storm especially in the last few years. In this project I undertook to use machine learning to predict the price of Bitcoin and Etherum. To do this I utilised a hybrid stacked model, that combined feature and target transforming models. To extrapolate the trend I used a LSTM network built using the Keras library, that was trained on features averaged over a large time window. While an XGboost model was trained on the residuals and features that encapsulated shorter time movements. In the end I was able to beat my baseline which predicated the price at time t+1 to be equal to the price at time t.

Currently working on utilising the Google Cloud platform in order to update the model every few days with new data, and retrain if necessary.

Sklearn, Keras, XGboost, Matplotlib, Google Cloud Platform
